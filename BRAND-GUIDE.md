## üöÄ Application Guidelines

### Logo Usage

#### Do's
- Use on solid backgrounds
- Maintain minimum clear space
- Keep proportions intact
- Use approved color combinations

#### Don'ts
- Don't stretch or distort
- Don't use on busy backgrounds
- Don't change colors arbitrarily
- Don't separate icon from text

### Color Usage

#### Backgrounds
- Primary: Black (#000000)
- Secondary: Dark Gray (#111111)
- Cards/Panels: Linear gradients with black base

#### Text
- Primary: White (#FFFFFF)
- Secondary: Light Gray (#CCCCCC)
- Accents: Neon colors for highlights

#### Interactive Elements
- Buttons: Neon colors with black text
- Links: Neon green (#00FF00)
- Hover states: Color shifts and glows
- Active states: Brighter neon variants

### Typography Hierarchy

#### Headings
- H1: 48px+, Weight 900, Uppercase, Letter-spacing
- H2: 32px+, Weight 800, Uppercase
- H3: 24px+, Weight 700, Uppercase
- H4: 18px+, Weight 600

#### Body Text
- Large: 18px, Weight 400
- Regular: 16px, Weight 400
- Small: 14px, Weight 400
- Caption: 12px, Weight 500

#### Special Text
- Numbers: JetBrains Mono, Weight 600
- Addresses: JetBrains Mono, Weight 400
- Code: JetBrains Mono, Weight 400

## üé® UI Components

### Buttons

#### Primary Button
- Background: Neon Green (#00FF00)
- Text: Black (#000000)
- Border: 2-3px solid black
- Hover: Glow effect + slight scale

#### Secondary Button
- Background: Black (#000000)
- Text: White (#FFFFFF)
- Border: 2-3px solid white
- Hover: Background to white, text to black

#### Danger Button
- Background: Neon Pink (#FF0080)
- Text: White (#FFFFFF)
- Border: 2-3px solid black
- Hover: Glow effect

### Cards
- Background: Linear gradient (black to dark gray)
- Border: 3px solid gray (#333333)
- Hover: Border color to neon green
- Shadow: None (sharp aesthetic)

### Forms
- Input Background: Black (#000000)
- Input Border: 2px solid white
- Focus: Border to neon green
- Text: White (#FFFFFF)
